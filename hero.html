¿Cómo oscurecer una imagen de fondo?

Para oscurecer una imagen de fondo en una sección hero, puedes utilizar el pseudo-elemento ::before con un color oscuro semitransparente.

El código CSS clave es:

            .hero::before {
                content: '';
                position: absolute;
                top: 0;
                left: 0;
                width: 100%;
                height: 100%;
                background: rgba(100, 0, 0, 1); /* Negro con 60% de opacidad */
                z-index: 1;
            }

Puedes ajustar el nivel de oscurecimiento modificando el valor de opacidad (0.6 en este ejemplo). Un valor más alto como 0.8 oscurecerá más la imagen, mientras que un valor más bajo como 0.4 la oscurecerá menos.

También puedes cambiar el color para lograr diferentes efectos. Por ejemplo:

  • rgba(0, 0, 0, 0.6) - Negro transparente (efecto de oscurecimiento)
  • rgba(30, 30, 46, 0.6) - Azul oscuro transparente
  • rgba(100, 0, 0, 0.6) - Rojo oscuro transparente
