Controles de Opacidad

Opacidad del overlay: 50%

¿Cómo funciona?

El control deslizante ajusta la opacidad de la capa oscura superpuesta sobre la imagen de fondo. Esto se logra mediante JavaScript que modifica el valor del canal alfa en el color RGBA.

El código JavaScript utilizado es:

// Obtener elementos del DOM
const overlay = document.getElementById('overlay');
const slider = document.getElementById('opacity-slider');
const valueDisplay = document.getElementById('opacity-value');

// Actualizar overlay cuando se mueve el slider
slider.addEventListener('input', function() {
  const opacity = this.value / 100;
  overlay.style.background = `rgba(0, 0, 0, ${opacity})`;
  valueDisplay.textContent = `${this.value}%`;
});

Puedes utilizar este mismo principio para crear overlays de diferentes colores simplemente cambiando los valores RGB en rgba(r, g, b, opacity).
